Download RedNotebook

Linux

  • Flatpak
  • Ubuntu PPA
  • Search for RedNotebook in your package manager (version might be outdated).

Windows

  • Important: Always backup your journal and close RedNotebook before upgrading.
  • Latest installer (no word clouds): rednotebook-***VERSION***.exe
  • Old installer with word clouds: rednotebook-1.15.exe
  • Portable version (may be outdated): PortableApps

Unfortunately, RedNotebook 2.x doesn't support word clouds on Windows (see also #415). Hopefully, this will be fixed in the near future. Currently, RedNotebook 1.15 is the latest version with word clouds on Windows.

MacOS

  • Install RedNotebook using Homebrew

Source

  • Download the latest source tarball, unpack it and change into the unpacked folder in a terminal. Then run:
./run
